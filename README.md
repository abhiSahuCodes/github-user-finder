# GitHub User Finder

Search any GitHub username and instantly see key profile details.

**Live Demo:** https://githubuser-finder-abhi.netlify.app/

## Features

- Search GitHub users by username
- Shows avatar, name, location, repos, followers, and bio
- Opens the userâ€™s website (if available) or GitHub profile link
- Mobile-responsive UI with inline loading + error states

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- GitHub REST API (`https://api.github.com/users/:username`)

## Run Locally

1. Clone the repo
2. Open `index.html` in your browser  

## Notes

- GitHub API rate limits apply (especially for unauthenticated requests).
